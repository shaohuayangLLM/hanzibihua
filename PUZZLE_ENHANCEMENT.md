## ✨ 新增功能

### 1. 错题本系统 📚

**功能亮点**:
- ✅ 自动记录错题到本地存储
- ✅ 错题计数和最后错误时间
- ✅ 用户错误答案历史追踪
- ✅ 掌握状态标记
- ✅ 错题复习模式

**实现细节**:

**文件**: `src/hooks/useWrongQuestions.ts`

```typescript
interface WrongQuestionRecord {
  questionId: string;
  question: PuzzleQuestion;
  wrongCount: number;        // 答错次数
  lastWrongTime: number;     // 最后答错时间
  userAnswers: string[];     // 错误答案历史
  correctAnswer: string;
  mastered: boolean;         // 是否已掌握
}
```

**核心API**:
- `addWrongQuestion()` - 自动添加/更新错题
- `markAsMastered()` - 标记为已掌握
- `getUnmasteredQuestions()` - 获取未掌握的错题
- `clearAllWrongQuestions()` - 清空错题本

**数据持久化**:
- 存储方式: `localStorage['puzzle-wrong-questions']`
- 自动保存: 每次变更后同步
- 跨会话: 页面刷新数据保持

**UI集成**:
1. **菜单页面**: 显示错题统计卡片
   - 错题总数
   - 未掌握数量
   - "开始复习"按钮

2. **结果页面**: 提示新增错题
   - 本次答错题目数
   - 快速复习入口

### 2. 智能提示系统 💡

**功能亮点**:
- ✅ 三级提示递进
- ✅ 思考币消耗机制
- ✅ 答对奖励思考币
- ✅ 智能排除法提示

**实现细节**:

**文件**: `src/hooks/useHintSystem.ts`

**提示级别**:

| 级别 | 名称 | 内容 | 消耗 |
|------|------|------|------|
| `none` | 无提示 | - | 0 币 |
| `basic` | 基础提示 | 题目自带hint | 0 币（免费） |
| `advanced` | 进阶提示 | 排除1个错误选项 | 1 币 |
| `full` | 完整提示 | 缩小到2个选项 | 2 币 |

**思考币机制**:
- **初始**: 10 个思考币
- **奖励**:
  - 无提示答对: +2 币
  - 使用提示答对: +1 币
- **消耗**:
  - 进阶提示: -1 币
  - 完整提示: -2 币

**智能提示生成**:
```typescript
// 进阶提示示例
"💡 这个不对：「树」，再想想其他选项"

// 完整提示示例
"🎯 答案在「林」和「森」之间"
```

**UI集成**:
1. **顶部导航**: 思考币余额显示（金币图标）
2. **答题区**: "使用提示"按钮
3. **提示卡片**: 蓝色背景，灯泡图标
4. **结果统计**: 显示提示使用次数

## 💡 技术亮点

### 1. 自定义 Hook 模式

**useWrongQuestions**:
- 完整的错题管理状态
- localStorage 自动同步
- 统计信息实时计算
- API 设计清晰简洁

**useHintSystem**:
- 提示级别状态管理
- 思考币消耗/奖励逻辑
- 智能提示生成算法
- 答题状态追踪

### 2. 数据持久化策略

**localStorage 结构**:
```typescript
{
  "puzzle-wrong-questions": WrongQuestionRecord[],
  // 每条记录包含:
  // - questionId
  // - question (完整题目对象)
  // - wrongCount (错误次数)
  // - userAnswers (历史答案)
  // - mastered (掌握状态)
}
```

**优势**:
- ✅ 无需后端，零成本
- ✅ 跨会话数据保持
- ✅ 隐私保护（本地存储）
- ✅ 即时读写，无延迟

### 3. 游戏化设计

**思考币经济系统**:
- **收入**: 答对题目（1-2 币）
- **支出**: 使用提示（1-2 币）
- **平衡**: 鼓励独立思考，适度提供帮助

**激励机制**:
- 🏆 完美通关（无提示全对）
- 🌟 错题攻克（复习后全对）
- 💪 持续进步（错误率下降）

### 4. 智能提示算法

**排除法提示**:
```typescript
// 随机排除一个错误选项
const wrongOptions = options.filter(opt => opt !== correctAnswer);
const randomWrong = wrongOptions[random];
return `💡 这个不对：「${randomWrong}」`;
```

**范围缩小提示**:
```typescript
// 保留正确答案 + 1个干扰项
return `🎯 答案在「${correctAnswer}」和「${keepOption}」之间`;
```
