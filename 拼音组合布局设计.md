# 拼音组合页面布局设计方案

## 方案一：经典表格

```
┌─────────────────────────────────────────────────────────────────┐
│  ← 声韵组合练习                                                  │
├─────────────────────────────────────────────────────────────────┤
│ [b p m f] [d t n l] [g k h] [j q x] [zh ch sh r] [z c s]     │
├─────────────────────────────────────────────────────────────────┤
│ 声母：b p m f | 点击卡片听朗读                                    │
├─────────────────────────────────────────────────────────────────┤
│  ┌───┬─────────┬─────────┬─────────┬─────────┐                 │
│  │韵母│   b     │   p     │   m     │   f     │                 │
│  ├───┼─────────┼─────────┼─────────┼─────────┤                 │
│  │ a │bā bǎ bà │pā pá pà │mā mǎ mà │fā fǎ fà │                 │
│  │   │八 爬 把 │趴 怕 爬 │妈 马 骂 │发 法 罚 │                 │
│  ├───┼─────────┼─────────┼─────────┼─────────┤                 │
│  │ o │    —    │    —    │   mō    │    —    │                 │
│  │   │         │         │ 摸       │         │                 │
│  ├───┼─────────┼─────────┼─────────┼─────────┤                 │
│  │ e │    —    │    —    │   me    │   fó    │                 │
│  │   │         │         │ 么       │ 佛       │                 │
│  └───┴─────────┴─────────┴─────────┴─────────┘                 │
│  ← 可横向滚动查看更多声母                                         │
└─────────────────────────────────────────────────────────────────┘

特点：
✓ 信息密度最高，一屏可见大量内容
✓ 表格式清晰，容易定位
✓ 固定表头，内容可横向滚动
✓ 点击单元格朗读/展开详情
✓ 无效组合用 — 表示
```

├───────────────────────────────────────────────────────┤
│  ┌───────────┐ ┌───────────┐ ┌───────────┐           │
│  │    a     │ │    ai    │ │    an    │           │
│  │  韵母    │ │  韵母    │ │  韵母    │           │
│  ├───────────┤ ├───────────┤ ├───────────┤           │
│  │bā 八     │ │bāi 逼   │ │bān 班   │           │
│  │bǎ 把     │ │bǎi 摆   │ │bǎn 板   │           │
│  │bà 爸     │ │bài 拜   │ │bàn 办   │           │
│  │          │ │          │ │          │           │
│  │🔊        │ │🔊        │ │🔊        │           │
│  └───────────┘ └───────────┘ └───────────┘           │
│  ┌───────────┐ ┌───────────┐ ┌───────────┐           │
│  │    ang   │ │    ao    │ │    e     │           │

## 方案四：分栏导航

```
┌───────────────────────────────────────────────────────┐
│  ← 声韵组合练习          [b p m f ▼]                  │
├──────────┬────────────────────────────────────────────┤
│          │ 声母：b p m f                              │
│  韵母列表 │──────────────────────────────────────────│
│          │                                            │
│  ▸ a     │  ┌────┬────┬────┬────┐                   │
│  ▸ o     │  │ b │ p │ m │ f │  韵母 a 的所有组合     │
│  ▸ e     │  ├────┼────┼────┼────┤                   │
│  ▸ i     │  │bā  │pā  │mā  │fā  │                   │
│  ▸ u     │  │八  │爬  │妈  │发  │                   │
│  ▸ ü     │  ├────┼────┼────┼────┤                   │
│  ▸ ai    │  │bǎ  │pá  │mǎ  │fǎ  │                   │
│  ▸ ei    │  │把  │怕  │马  │法  │                   │
│  ▸ ao    │  ├────┼────┼────┼────┤                   │
│  ▸ ou    │  │bà  │pà  │mà  │fà  │                   │
│  ▸ ...   │  │爸  │怕  │骂  │罚  │                   │
│          │  └────┴────┴────┴────┘                   │
│ [搜索]   │  🔊 点击朗读                                │
│          │                                            │
│          │  ┌────┬────┬────┬────┐                   │
│          │  │ b │ p │ m │ f │  韵母 o 的所有组合     │
│          │  ├────┼────┼────┼────┤                   │
│          │  │ —  │ —  │mō  │ —  │                   │
│          │  │    │    │摸  │    │                   │
│          │  └────┴────┴────┴────┘                   │
│          │                                            │
└──────────┴────────────────────────────────────────────┘

特点：
✓ 左侧固定韵母导航
✓ 右侧展示选中韵母的组合
✓ 切换流畅，无需翻页
✓ 适合内容较多时的导航
```

## 对比总结

| 方案 | 信息密度 | 视观效果 | 交互性 | 适用场景 |
|------|---------|---------|--------|---------|
| 经典表格 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ | 快速查阅对比 |
| 列表式 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 逐韵母学习 |
| 卡片瀑布流 | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 探索浏览 |
| 分栏导航 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 系统学习 |

**推荐**：对于拼音组合学习，建议使用 **方案一（经典表格）** 或 **方案四（分栏导航）**，因为：
1. 需要横向对比不同声母的组合
2. 表格式最直观，类似教材
3. 可以快速定位和查看
