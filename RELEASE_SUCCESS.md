### 标题
🎉 K12 汉字图谱 2.0 Beta 版正式发布！

### 正文
经过 3 天的开发，K12 汉字图谱 2.0 Beta 版正式发布！

**新功能亮点**：
🎓 **6 种关系类型**：形近字、同部首、组词、结构、多音字、例句
🧠 **智能推荐**：根据学习记录推荐下一个字
👣 **学习足迹**：记录完整的学习路径和可视化
🎯 **首次引导**：7 步交互式引导，零门槛上手
📱 **移动适配**：手机、平板、电脑都能流畅使用

**访问地址**：
https://ainside.cn/k12/character-graph

**使用方法**：
1. 观看首次使用引导
2. 从"天"字开始探索
3. 单击查看详情，双击切换中心
4. 使用左侧筛选器切换关系类型
5. 查看智能推荐，点击快速学习

**反馈渠道**：
如有任何问题或建议，欢迎通过 GitHub Issues 或 Email 反馈。
