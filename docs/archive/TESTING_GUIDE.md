# 汉字图谱 2.0 测试指南

## 🚀 启动应用

开发服务器地址: **http://localhost:8080/k12/**

导航到图谱页面: **http://localhost:8080/k12/character-graph**

---

## ✅ 功能测试清单

### 1. 基础图谱显示 ✓

**测试步骤:**
1. 打开 http://localhost:8080/k12/character-graph
2. 应该看到中心字"天"（蓝色大节点）
3. 周围有若干关联字（形近字、部首、组词等）

**验证点:**
- [ ] 中心节点显示正确（天，大字体，蓝色边框）
- [ ] 关联节点显示正确（拼音、教材标签）
- [ ] 节点之间有连线
- [ ] 可以缩放（滚轮）、拖拽画布
- [ ] MiniMap 显示在右下角

---

### 2. 汉字详情弹窗 ✓

**测试步骤:**
1. **单击**任意汉字节点（如"大"）
2. 应该弹出详情窗口

**验证点:**
- [ ] 显示拼音、笔画数、部首
- [ ] 显示组词（至少3个）
- [ ] 显示例句
- [ ] 有"标记已学"按钮
- [ ] 点击 ✕ 或按 Esc 可关闭

---

### 3. 学习状态标记 ✓

**测试步骤:**
1. 单击"大"字节点
2. 点击"标记已学"按钮
3. 关闭弹窗

**验证点:**
- [ ] 节点变为绿色边框
- [ ] 节点左上角显示绿色 ✓ 图标
- [ ] 左侧"学习统计"中"已学"数量 +1
- [ ] 刷新页面后状态保持（localStorage）

---

### 4. 切换中心节点 ✓

**测试步骤:**
1. **双击**任意关联字节点（如"夫"）
2. 节点应切换为中心

**验证点:**
- [ ] "夫"字变为中心节点（大字体，蓝色）
- [ ] 周围显示"夫"的关联字
- [ ] 原中心"天"变为普通节点
- [ ] 学习足迹中新增记录："15:XX 夫 ← 天"
- [ ] 当前路径显示："天 → 夫"

---

### 5. 关系筛选 ✓

**测试步骤:**
1. 在左侧控制面板中
2. 关闭"形近字"开关
3. 关闭"组词关系"开关

**验证点:**
- [ ] 形近字节点消失
- [ ] 组词关系节点消失
- [ ] 仅显示"同部首"和"笔画相近"的字
- [ ] 重新打开开关后节点恢复

---

### 6. 教材筛选 ✓

**测试步骤:**
1. 顶部选择"一年级下册"
2. 观察节点变化

**验证点:**
- [ ] （注意：当前筛选不会过滤节点，只影响新构建的图谱）
- [ ] 节点标签显示正确（一下/二上等）

---

### 7. 搜索功能 ✓

**测试步骤:**
1. 在顶部搜索框输入"春"
2. 按 Enter 键

**验证点:**
- [ ] "春"字成为新的中心节点
- [ ] 周围显示"春"的关联字
- [ ] 学习路径中新增："XX → 春"

---

### 8. 学习足迹追踪 ✓

**测试步骤:**
1. 依次双击：天 → 大 → 太 → 阳
2. 观察左侧"学习足迹"

**验证点:**
- [ ] 最近记录显示 4 条（倒序）
- [ ] 每条记录显示时间、字、来源（如"15:30 太 ← 大"）
- [ ] 当前路径显示："天 → 大 → 太 → 阳"
- [ ] 统计信息正确：总记录 4 条，不同字 4 个

---

### 9. 路径重置与清空 ✓

**测试步骤:**
1. 点击"重置"按钮（当前路径下方）
2. 点击"清空所有记录"按钮

**验证点:**
- [ ] 重置后：当前路径清空，但历史记录保留
- [ ] 清空后：所有记录消失，统计归零
- [ ] 所有已学节点恢复为未学状态

---

### 10. 移动端响应式 ✓

**测试步骤（桌面浏览器）:**
1. 打开浏览器开发者工具（F12）
2. 切换到移动端视图（Ctrl+Shift+M 或点击设备图标）
3. 选择 iPhone 12 Pro 或类似设备

**验证点:**
- [ ] 左侧控制面板隐藏
- [ ] 顶部显示菜单按钮（☰）
- [ ] 点击菜单按钮打开抽屉
- [ ] 抽屉中显示控制面板 + 学习足迹
- [ ] 搜索框变窄（w-20）
- [ ] 教材筛选器隐藏

---

### 11. 数据持久化 ✓

**测试步骤:**
1. 标记几个字为"已学"
2. 双击几次切换中心节点
3. 刷新页面（F5）

**验证点:**
- [ ] 已学状态保持（节点仍为绿色）
- [ ] 学习足迹保留（历史记录仍在）
- [ ] 当前路径保留

**LocalStorage 验证:**
- 打开浏览器控制台 → Application → Local Storage
- 应该看到 `learning_records` 和 `current_learning_path` 键

---

## 🐛 已知问题/限制

1. **教材筛选不实时过滤**: 切换教材后需要切换中心字才生效
2. **词语节点未实现**: 双击词语节点暂无反应
3. **引导动画缺失**: 首次使用无引导提示
4. **路径可视化**: 画布上暂无高亮路径连线

## 🔧 调试技巧

### 查看学习记录（浏览器控制台）
```javascript
// 查看所有学习记录
JSON.parse(localStorage.getItem('learning_records'))

// 查看当前路径
JSON.parse(localStorage.getItem('current_learning_path'))

// 清空所有数据
localStorage.clear()
```

### React DevTools
- 安装 React DevTools 扩展
- 查看组件树: CharacterGraph → useLearningPath
- 查看 state: records, currentPath
