## 🎉 核心功能

### 1. 教材驱动的汉字关系图谱
- **5 种关系类型**: 形近字、同部首、组词、笔画数、自定义关系
- **教材筛选**: 支持一年级上/下册、二年级上/下册、常用字、部首表
- **关系筛选**: 可独立开关每种关系类型
- **圆形布局**: 中心字 + 关联字的直观展示
- **交互操作**:
  - 单击节点查看详情（拼音、释义、组词、例句）
  - 双击节点切换为中心字
  - 搜索框快速跳转到任意汉字

### 2. 智能学习路径追踪
- **学习足迹时间线**: 记录每个学习的汉字及探索路径
- **路径可视化**:
  - 查看完整学习路径（天 → 大 → 太 → 阳）
  - 高亮路径上的节点和边
  - 路径统计（汉字数量、学习时长）
- **学习状态管理**:
  - 已学 / 已掌握标记
  - 重置路径 / 清空所有记录
  - localStorage 本地持久化

### 3. 个性化智能推荐
- **推荐算法**: 基于多维度加权评分
  - 关系强度 (形近字、同部首、组词)
  - 未学字优先 (50% 权重)
  - 同教材优先 (30% 权重)
  - 字频统计 (20% 权重)
- **推荐卡片**: 显示前 3 名推荐汉字
  - 排名徽章 (金/银/铜)
  - 推荐理由 (形近字/同部首/组词)
  - 一键跳转学习

### 4. 首次使用引导
- **7 步交互式引导**: 从欢迎到操作说明
- **脉冲高亮效果**: 引导用户关注重点区域
- **进度指示器**: 清晰展示引导进度
- **localStorage 记忆**: 自动判断是否首次使用
- **随时重新打开**: 顶部帮助按钮可重新查看引导

### 5. 移动端完美适配
- **响应式设计**: 桌面端 / 移动端自动适配
- **移动端侧边栏**: Sheet 抽屉式面板
- **触控优化**: 手势操作流畅
- **画布缩放**: 支持双指缩放、拖拽

### 6. 数据持久化
- **localStorage 存储**: 学习记录、路径、配置自动保存
- **跨会话保持**: 关闭浏览器后数据不丢失
- **自动加载**: 打开页面自动恢复上次状态

## 🚀 技术亮点

### 架构设计
- **React 18 + TypeScript**: 类型安全，开发体验优秀
- **ReactFlow v11**: 专业级图谱渲染库
- **Vite**: 极速构建和热更新
- **shadcn/ui**: Apple 风格 UI 组件库
- **Tailwind CSS**: 实用优先的 CSS 框架

### 性能优化
- **useMemo / useCallback**: 避免不必要的重新渲染
- **节点数量限制**: 最多 30 个节点，保持流畅
- **圆形布局算法**: 均匀分布，视觉美观
- **虚拟化渲染**: ReactFlow 内置优化

### 代码质量
- **15+ TypeScript 接口**: 完整的类型定义
- **自定义 Hook**: useLearningPath, useTutorial
- **组件复用**: GraphSidebarContent 桌面端 / 移动端共用
- **关注点分离**: 数据层 / 业务逻辑 / UI 展示清晰分离

## 🐛 已知问题

### 非阻塞性问题
1. **Bundle 体积较大**: 1.1 MB (gzip: 358 KB)
   - 原因: ReactFlow 库较大
   - 计划: 代码分割、动态导入

2. **路径高亮性能**: 节点数 > 50 时可能卡顿
   - 缓解: 限制最多 30 个节点
   - 计划: 虚拟化渲染优化

### 功能限制
1. **只支持一年级汉字**: 二年级数据尚未录入
2. **无云端同步**: 数据仅存储在本地
3. **无导出功能**: 无法分享学习报告
