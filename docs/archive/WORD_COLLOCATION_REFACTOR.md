# 词语搭配功能问题分析与重构方案

## 📋 当前实现问题诊断

### 问题一：搭配概念混乱 ❌

#### "名词+名词"分类错误

**错误案例1** (wc027):
```typescript
{
  word: "树叶",
  category: "名词+名词",
  correct: "绿色",  // ❌ "绿色"是形容词，不是名词！
  options: ["绿色", "飞", "飘", "一片"],  // 词性混乱
  examples: ["树叶绿色的", ...]  // ❌ 语法错误！
}
```

**问题**：
- "绿色"是形容词，不是名词
- 例句"树叶绿色的"语法不通顺
- 应该是：**形容词修饰名词**，即"绿色的树叶"

**错误案例2** (wc028):
```typescript
{
  word: "鱼儿",
  category: "名词+名词",
  correct: "水里",  // ❌ "水里"是地点短语，不是名词！
  options: ["水里", "游", "跳", "一条"],  // 词性混乱
}
```

**问题**：
- "水里"是方位短语（地点），不是纯名词
- 这是**主谓短语**或**名词+地点**，而非名词+名词

---

### 问题二：选项设计不合理 ❌

#### 同一题目选项词性不一致

**案例：wc027**
```
题目：树叶 ___
选项：["绿色"(形容词), "飞"(动词), "飘"(动词), "一片"(量词)]
```

**问题**：
- 选项包含4种不同词性的词
- 学生无法学到词语搭配规律
- 正确答案应该从**同类词性**中选择

**正确设计应该是**：
```
题目：树叶 ___
选项：["飘落"(动词), "摇动"(动词), "变黄"(动词), "飞舞"(动词)]
// 或者改为：___ 树叶
选项：["绿色"(形容词), "金黄"(形容词), "茂密"(形容词), "枯黄"(形容词)]
```

---

### 问题三：例句质量低 ❌

#### 语法错误和不完整

**错误例句**：
1. "树叶绿色的" ❌ - 主谓不完整，缺少谓语
2. "鱼儿在水里" ❌ - 句子不完整，缺少谓语
3. "白云朵" ❌ - 词序错误，应该是"白白的云朵"

**正确例句应该是**：
1. "绿色的树叶在风中摇摆" ✅
2. "鱼儿在水里快乐地游来游去" ✅
3. "天上飘着朵朵白云" ✅

### 问题四：搭配类型不全 ❌

#### 当前分类

```typescript
const COLLOCATION_CATEGORIES = [
  "全部",
  "形容词+名词",   // ✅ 有
  "动词+名词",     // ✅ 有
  "名词+名词",     // ❌ 错误理解
  "数量+名词",     // ✅ 有
  "形容词+动词",   // ✅ 有，但应该是"副词+动词"
];
```

#### 缺少的重要类型

1. **副词+动词** - 如：慢慢地走、快快地跑、悄悄地说
2. **副词+形容词** - 如：非常美丽、特别高兴、十分聪明
3. **动词+动词** - 如：跑来跑去、看了看、想一想
4. **真正的名词+名词合成词** - 如：教师节、儿童节、图书馆、火车站

---

### 问题五：教学价值不足 ❌

1. **缺少搭配规则说明**
   - 没有告诉学生为什么这样搭配
   - 缺少词性概念的解释

2. **缺少常见错误提示**
   - 没有标注学生容易混淆的搭配
   - 缺少对比错误案例

3. **缺少应用场景**
   - 例句脱离实际语境
   - 没有生活化的应用

## 🔧 重构方案

### 1. 重新定义搭配类型

#### 新的分类体系

```typescript
export const NEW_COLLOCATION_CATEGORIES = {
  // 定语类（修饰名词）
  ADJECTIVE_NOUN: {
    id: "adj-noun",
    name: "形容词+名词",
    description: "形容词修饰名词，描述事物的性质、状态",
    example: "美丽的花朵、蓝蓝的天空",
  },

  NOUN_QUANTIFIER: {
    id: "quan-noun",
    name: "量词+名词",
    description: "表示事物的数量",
    example: "一朵花、两本书、三只鸟",
  },

  // 谓语类（动作行为）
  VERB_OBJECT: {
    id: "verb-obj",
    name: "动词+宾语",
    description: "动词后面接宾语，表示动作的对象",
    example: "读书、唱歌、踢足球",
  },

  // 状语类（修饰动词/形容词）
  ADVERB_VERB: {
    id: "adv-verb",
    name: "副词+动词",
    description: "副词修饰动词，说明动作的方式、程度",
    example: "慢慢地走、认真地听、快快地跑",
  },

  ADVERB_ADJECTIVE: {
    id: "adv-adj",
    name: "副词+形容词",
    description: "副词修饰形容词，表示程度",
    example: "非常美丽、特别高兴、十分聪明",
  },

  // 复合词类
  NOUN_COMPOUND: {
    id: "noun-compound",
    name: "名词合成词",
    description: "两个名词组成新词",
    example: "教师节、图书馆、火车站",
  },

  // 特殊类
  VERB_COMPLEMENT: {
    id: "verb-comp",
    name: "动词+补语",
    description: "动词后面接补语，说明结果、趋向",
    example: "看完、走进来、跑出去",
  },
};
```

### 2. 统一选项设计原则

#### 原则一：同类词性选项

**坏例子** ❌
```typescript
{
  word: "美丽",
  options: ["花朵"(名词), "跑"(动词), "大声"(副词), "一个"(量词)],
}
```

**好例子** ✅
```typescript
{
  word: "美丽的",
  options: ["花朵"(名词), "树木"(名词), "蝴蝶"(名词), "小鸟"(名词)],
}
```

#### 原则二：难度递进选项

```typescript
options: [
  "花朵",    // ✅ 常见搭配
  "公园",    // ✅ 正确但不常见
  "星星",    // ✅ 正确但较抽象
  "风景",    // ✅ 正确答案
]
```

### 3. 例句质量标准

#### 标准一：完整的句子

❌ "树叶绿色的"
✅ "树上的叶子绿色的，很好看。"

#### 标准二：生活化场景

❌ "美丽的花朵"（孤立词组）
✅ "春天到了，公园里开满了美丽的花朵。"

#### 标准三：多样化句式

```typescript
examples: [
  "春天到了，公园里开满了美丽的花朵。",      // 陈述句
  "你看，那些美丽的花朵多么鲜艳啊！",        // 感叹句
  "小朋友们都喜欢美丽的花朵。",              // 主谓宾句
]
```

### 4. 数据结构优化

#### 新的数据结构

```typescript
export interface ImprovedCollocationExercise {
  id: string;

  // 基础信息
  category: string;              // 搭配类型
  categoryDescription: string;   // 类型说明

  // 题目信息
  left: string;                  // 前半部分词语
  leftPinyin: string;            // 拼音
  leftPos: string;               // 词性（part of speech）

  right: string;                 // 后半部分词语（空白处）
  rightPos: string;              // 词性

  connector?: string;            // 连接词（如"的"、"地"、"得"）

  // 答案和选项
  correct: string;               // 正确答案
  options: string[];             // 选项（同一词性）

  // 教学辅助
  examples: {
    sentence: string;            // 完整句子
    highlight: string;           // 高亮部分
  }[];

  tip?: string;                  // 搭配规律提示
  commonMistakes?: string[];     // 常见错误

  // 难度和标签
  difficulty: "easy" | "medium" | "hard";
  tags: string[];                // 如：["一年级上", "常用", "重点"]
}
```
