## 📢 发布说明

### 给用户的说明

**汉字图谱 2.0 Beta 版现已发布！**

这是一个帮助小朋友探索汉字关系的学习工具。通过图谱，可以看到汉字之间的形近字、同部首、组词、例句、结构、多音字等 6 种关系。

**主要特点**：
- 🎓 教材驱动：支持一年级上/下册筛选
- 🧠 智能推荐：根据学习记录推荐下一个字
- 👣 学习足迹：记录完整的学习路径
- 🎯 首次引导：7 步交互式引导，降低使用门槛
- 📱 移动适配：手机、平板、电脑都能用

**如何使用**：
1. 访问 https://ainside.cn/k12/character-graph
2. 观看首次使用引导
3. 从"天"字开始探索
4. 单击查看详情，双击切换中心
5. 使用左侧筛选器切换关系类型

**反馈渠道**：
- GitHub Issues: [项目仓库地址]
- Email: [联系邮箱]

### 给开发者的说明

**技术亮点**：
- TypeScript 完整类型定义

## ⚠️ 已知问题

### 非阻塞性问题
1. **Bundle 体积较大**: 1.1 MB (gzip: 359 KB)
   - 原因: ReactFlow 库较大
   - 计划: 代码分割、动态导入

2. **二年级数据不完整**: 部分二年级汉字数据未录入
   - 影响: 筛选"二年级"教材时字数较少
   - 计划: 后续版本补充

3. **首次使用引导**: 在低性能设备上可能略显卡顿
   - 缓解: 优化动画性能
   - 计划: 使用 requestAnimationFrame

### 功能限制
1. **只支持一年级为主**: 二年级数据待补充
2. **无云端同步**: 数据仅存储在本地
3. **无导出功能**: 无法分享学习报告
4. **无动画效果**: 节点切换无动画
