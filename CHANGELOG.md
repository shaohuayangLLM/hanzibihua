# 更新日志 (Changelog)

所有重要变更都将记录在此文件中。

## [1.0.0] - 2025-01-23

### 🎉 首个正式版本发布

### ✨ 新增功能

#### 语文学习模块

**多音字练习**
- 38个常见多音字填空练习
- 每个多音字展示不同读音和例词
- 句子填空交互式练习
- 实时反馈（正确/错误提示）
- 全部多音字概览网格
- 支持快速跳转到任意多音字

**形近字学习**
- 52组形近字对比学习
- 每组展示字形、拼音、释义、例词
- 详细的辨析提示
- 支持点击切换学习
- 全部形近字概览网格

**偏旁学习**
- 常用偏旁部首练习
- 偏旁名称、例字展示
- 互动式学习体验

**拼音基础**
- 23个声母、24个韵母、16个整体认读音节
- 详细的拼音教学卡片
- 音频朗读功能
- 读音测试

**拼音测试**
- 综合测试模式
- 鼻音对比专项（前/后鼻音）
- 舌音对比专项（平/翘舌音）
- 智能题目生成
- 实时评分统计

#### 数学学习模块

**苏教版一年级上册完整内容**
- 数与代数：10以内、20以内数的认识
- 20以内进位加法
- 20以内退位减法
- 分与合
- 图形的认识
- 队列问题

**练习模式**
- 每个知识点专项练习
- 即时反馈
- 支持重做和继续

**测试模式**
- 综合测试
- 单元测试
- 限时测试
- 详细成绩分析

**知识点讲解**
- 图文并茂的知识点讲解
- 例题演示
- 配套练习

### 📚 数据内容

**多音字数据库**
- 地、着、数、长、只、觉、种、为、分、没、得、还、干、都、乐、好、空、相、行、少、背、当、看、处、传、弹、会、朝、转、血、假、难、量、盛、教、漂、奇、兴、重、降、差、参、和

  - 等等...

### 🛠️ 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite 5
- **路由**: React Router v6
- **样式**: Tailwind CSS + shadcn/ui
- **动画**: HanziWriter (笔画动画)
- **状态管理**: useState, useReducer
- **PWA**: vite-plugin-pwa
- **部署**: Vercel (portfolio-2025 仓库)
