**功能增强**:

   - 新增 `selectedGroup` 支持 `InitialGroup | "all"` 类型
   - "所有声母"按钮使用蓝色渐变（与其他分组区分）
   - 保留原有 6 个分组按钮（绿色渐变）

2. ✅ **数据聚合**:
   - `selectedGroup === "all"` 时:
     - 聚合所有声母: `INITIAL_GROUPS.flatMap(group => getGroupInitials(group))`
     - 合并所有组数据:
       ```typescript
       INITIAL_GROUPS.reduce((acc, group) => {
         const groupData = getGroupData(group);
         for (const [final, items] of Object.entries(groupData)) {
           if (!acc[final]) acc[final] = [];
           acc[final].push(...items);
         }
         return acc;
       }, {} as CombinationData)
       ```
   - 否则使用 `getGroupData(selectedGroup)`

3. ✅ **UI 更新**:
   - 添加"所有声母"按钮（放在最前面）
   - 高亮状态: 蓝色渐变 vs 绿色渐变
   - 悬停效果统一

## 技术亮点

### 1. 数据处理
- **扁平化算法**: 二维表格 → 一维卡片列表
- **韵母顺序保留**: 按 `ALL_FINALS` 数组顺序遍历
- **有效性过滤**: 只显示有声调数据的组合

### 2. 性能优化
- **useMemo**: 缓存 `combinationList`，避免重复计算
- **键值优化**: 使用 `${initial}-${final}` 作为唯一 key
- **懒加载**: 音频按需播放，不预加载

### 3. 响应式设计
- **Tailwind 断点**: `sm:`, `lg:`, `xl:` 精细控制
- **字体缩放**: 移动端 `text-xs sm:text-sm`
- **触控优化**: `active:scale-95` 提供触控反馈

### 4. 用户体验
- **渐进增强**: 先显示静态卡片，后加载音频
- **错误降级**: 阿里云 TTS 失败 → Web Speech API
- **视觉反馈**: 悬停、点击、播放状态清晰可见
